# status-challenge

Smart contract challenge for [status.im](https://status.im).

### How to use?

1. Install dependencies: `npm install`
2. Deploy to
    - Ganache: `npm run deploy`
    - Rinkeby: `npm run deploy-rinkeby`
    - Mainnet: `npm run deploy-mainnet`
